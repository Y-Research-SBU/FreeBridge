# Contributing to FreeBridge

Thank you for your interest in FreeBridge.

This repository is maintained as a research codebase for single-cell latent transition modeling. Contributions that improve reproducibility, documentation, data preprocessing, or evaluation scripts are welcome.

## Pull Requests

1. Fork the repository and create a branch from `main`.
2. Keep changes focused and documented.
3. If you change training, evaluation, or data-processing behavior, include a short note explaining the expected effect.
4. Run basic syntax checks before submitting.

## Issues

Please include enough information to reproduce the issue, including:

- command used
- configuration file or Hydra overrides
- environment details
- relevant traceback or logs
